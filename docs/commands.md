# Quick Commands

## Setup
```bash
make setup      # Create database
```

## Data
```bash
make etl        # Extract, transform, and load data
make sql        # Open SQLite console
```

## Analysis
```bash
make nb         # Launch Jupyter notebooks
```

## Development
```bash
make check      # Lint code
make qa         # Run all checks
make clean      # Clean cache
```

## Common workflow
```bash
make setup      # First time
make etl        # Get and load data
make nb         # Analyze
```
